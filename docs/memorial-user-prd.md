# 纪念馆 — 用户端产品需求文档 (PRD)

## 一、产品概述

### 1.1 产品定位
面向逝者家属和亲友的线上纪念馆平台，用户可以创建和管理逝者纪念页、发布留言寄语、献花致敬、浏览相册视频。同时支持微信小程序端，方便用户随时缅怀纪念。

### 1.2 用户角色

| 角色 | 说明 |
|------|------|
| 家属用户 | 逝者直系亲属，拥有纪念页的管理权限（编辑信息、上传相册视频、管理留言） |
| 访客用户 | 亲友或扫二维码访客，可浏览纪念页、留言寄语、献花 |

### 1.3 端覆盖
- **微信小程序**（主端）：覆盖扫码场景和日常访问
- **H5 网页**（辅端）：分享到朋友圈/微信群的链接直接打开

> 技术方案建议采用 **uni-app**，一套代码编译为小程序 + H5，降低开发和维护成本。

## 二、功能模块

### 2.1 用户注册与登录

#### 微信小程序端
- 微信一键登录（wx.login → 获取 openid → 自动注册/登录）
- 首次登录自动创建账号，无需额外注册流程
- 登录后获取手机号（可选，用于订单通知）

#### H5 端
- 微信授权登录（OAuth2.0 网页授权）
- 手机号 + 验证码登录

### 2.2 首页

| 功能 | 说明 |
|------|------|
| 搜索逝者 | 按姓名搜索已公开的纪念页 |
| 最近纪念 | 展示用户最近访问的纪念页（本地缓存） |
| 机构入口 | 展示合作机构列表/入口（殡仪馆、公墓等） |
| 服务说明 | 平台服务介绍、套餐说明入口 |

### 2.3 纪念页（核心页面）

纪念页是整个产品的核心，家属和访客都在此完成主要操作。

#### 页面布局（自上而下）

| 区域 | 内容 |
|------|------|
| 顶部 | 逝者头像/照片、姓名、生卒年月 |
| 纪念文 | 家属撰写的纪念文章（富文本，含图文） |
| 相册 | 逝者生活照片瀑布流/九宫格展示，点击查看大图 |
| 视频 | 逝者相关视频列表，点击播放 |
| 留言寄语 | 已审核通过的留言列表，支持发表新留言 |
| 献花致敬 | 鲜花展示区 + 献花入口，显示献花总数 |
| 统计信息 | 访问量、留言数、献花数 |
| 分享按钮 | 生成小程序码/分享卡片，便于传播 |

#### 纪念页访问方式
- **扫码访问**：扫描后台生成的二维码，直接打开对应纪念页
- **搜索访问**：在首页搜索逝者姓名
- **分享访问**：通过微信分享卡片/链接进入

### 2.4 留言寄语

| 功能 | 说明 |
|------|------|
| 查看留言 | 浏览该逝者下已审核通过的留言列表 |
| 发表留言 | 填写留言人姓名 + 留言内容，提交后进入后台审核 |
| 字数限制 | 留言内容最多 500 字 |
| 提交反馈 | 提交后提示"留言已提交，审核通过后将展示" |

> 留言需后台审核后才展示，防止不良内容。

### 2.5 献花致敬

| 功能 | 说明 |
|------|------|
| 鲜花选择 | 提供菊花、百合、康乃馨、玫瑰等鲜花类型 |
| 献花操作 | 选择鲜花类型 → 确认献花 → 献花动画 → 计数+1 |
| 献花记录 | 在纪念页展示献花总数和类型分布 |
| 限制 | 同一用户对同一逝者每天可献花 3 次 |

### 2.6 相册

| 功能 | 权限 | 说明 |
|------|------|------|
| 浏览相册 | 所有人 | 查看逝者相册照片，支持左右滑动浏览大图 |
| 上传照片 | 家属 | 从手机相册选择照片上传，单次最多 9 张 |
| 删除照片 | 家属 | 长按/点击管理进入删除模式 |

### 2.7 视频

| 功能 | 权限 | 说明 |
|------|------|------|
| 播放视频 | 所有人 | 点击播放逝者相关视频 |
| 上传视频 | 家属 | 录制或从相册选择视频上传，单个视频 ≤ 50MB |
| 删除视频 | 家属 | 在管理页面删除视频 |

### 2.8 纪念页管理（家属专属）

家属用户在「我的」页面中管理自己创建的纪念页。

| 功能 | 说明 |
|------|------|
| 创建纪念页 | 填写逝者信息（姓名、性别、出生/死亡日期、纪念标题、纪念文章）→ 选择套餐 → 提交 |
| 编辑纪念页 | 修改逝者基本信息和纪念内容 |
| 上传相册 | 在纪念页管理中上传/删除照片 |
| 上传视频 | 在纪念页管理中上传/删除视频 |
| 管理留言 | 查看留言列表，可删除不当留言（家属维度） |
| 生成二维码 | 生成纪念页专属二维码，可保存到手机相册 |
| 分享纪念页 | 生成微信分享卡片 |

### 2.9 订单与服务套餐

| 套餐 | 功能 | 定价 |
|------|------|------|
| 基础纪念 | 纪念页 + 留言 + 献花 + 5张相册 | 免费 |
| 高级纪念 | 基础功能 + 50张相册 + 3个视频 + 自定义纪念页样式 | ¥99/年 |

2. 支付成功 → 套餐权益即时生效

### 2.10 个人中心

| 功能 | 说明 |
|------|------|
| 我的纪念页 | 我创建的纪念页列表，点击进入管理 |
| 我的留言 | 我发表的留言记录及审核状态 |
| 我的献花 | 我的献花记录 |
| 我的订单 | 订单列表及支付状态 |

## 四、非功能性需求

### 4.1 性能
- 纪念页首屏加载 ≤ 2 秒
- 图片采用 CDN + 懒加载 + 缩略图，大图按需加载
- 视频采用流式播放，不预加载

### 4.2 安全
- 留言内容后台审核机制，防止违规内容
- 献花频率限制（IP + 用户ID 双重限制）
- 图片上传格式和大小校验（≤ 5MB/张）
- 视频上传大小校验（≤ 50MB/个）
- 敏感操作（删除纪念页、退款）需二次确认

### 4.3 兼容性
- 微信小程序：基础库 ≥ 2.20.0
- H5：微信内置浏览器、Safari、Chrome 主流版本
- 适配 iPhone SE ~ iPhone 15 Pro Max、主流安卓机型

### 4.4 分享与传播
- 小程序分享：支持分享到聊天、朋友圈
- 二维码：生成带逝者信息的小程序码，可保存图片
- H5 分享：自定义分享卡片（标题、描述、缩略图）

## 五、技术方案建议

| 项目 | 方案 |
|------|------|
| 前端框架 | uni-app (Vue 3 + TypeScript) |
| UI 组件库 | uni-ui + 自定义组件 |
| 状态管理 | Pinia |
| 请求封装 | uni.request 二次封装，统一拦截鉴权和错误处理 |
| 小程序登录 | wx.login + 后端解密 openid |
| 支付 | 微信小程序支付（wx.requestPayment） |
| 图片上传 | uni.chooseImage + 后端接口 |
| 视频上传 | uni.chooseVideo + 分片上传 |
| 二维码生成 | 后端生成小程序码（wxacode.getUnlimited） |

## 六、与后台管理系统的关系

| 用户端操作 | 后台管理对应 |
|------------|------------|
| 创建纪念页 → 审核通过后展示 | 逝者管理页面可查看/编辑 |
| 发表留言 → 进入审核队列 | 留言管理页面审核/拒绝 |
| 献花 → 实时记录 | 献花记录页面可查看/删除 |
| 下单支付 → 订单生成 | 订单管理页面可查看/管理 |
| 上传相册/视频 | 后台可查看/管理媒体资源 |
| 访问纪念页 → 统计+1 | 统计管理页面查看访问数据 |
